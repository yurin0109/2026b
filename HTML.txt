- HTML : 하이퍼텍스트 마크업 언어 
- 마크업 : <마크업> 텍스트 </마크업>
- 마크업 속성 : <마크업 속성명="속성값"> </마크업>
- 마크업 종류
    0. <html> HTML전체
    0. <head> HTML문서설정 , <title> 문서제목
    0. <body> HTML본문
    1. <h1> ~ <h6>                  : 제목 
    2. <p>                          : 문단 
    3. <br />                       : 줄바꿈
    4. <hr />                       : 가로선 
    5. <span>                       : 인라인 구역
    6. <div>                        : 블럭 구역
    7. <ol> , <ul>                  : 목록
    * <li>                          : ol 또는 ul 마크업 안에 항목
    8. <a href="링크주소/파일경로">  : 하이퍼링크 
    9. <img src="파일경로" />       : 이미지
    10. <audio src="파일경로"> , <video src="파일경로"> : 오디오/비디오 
    * controls속성 : 컨트롤러 제공
    * autoplay속성 : 자동실행(브라우저정책상 차단)
    * muted속성 : 음소거 제공
    11. <iframe> : 외부 컨텐츠를 포함( 퍼가기 )

    12. <input />  : 입력 상자
        * 주요속성
        1. type = "text"            : 텍스트
        2. type = "password"        : 비밀번호
        3. type = "file"            : 첨부파일 
        4. type = "datetime-local"  : 날짜/시간
        5. type = "radio"           : 라디오(단일선택)
        6. type = "checkbox"        : 체크박스(복수선택)
        7. type = "submit"          : 폼전송
        8. type = "button"          : 일반버튼 *추후에 JS 연동*
        9. value = "입력된값"        : 입력상자에 입력된값 
        10. disabled                : 사용금지
        11. readonly                : 읽기모드      
        12. placeholder = "입력가이드" : 입력전 가이드텍스트
        13. name = "식별이름"          : 마크업 식별이름 
    13. <textarea>                  : 긴글 입력 상자
        * rows = "행텍스트수" , cols ="열텍스트수" 
    14. <select >                   : 목록 상자
        * <option>                  : 목록 항목
    15. <button> : 버튼             : 일반버튼 *추후에 JS 연동*
